Support

使用支援

好記適用於 iOS 18 以上版本。先從下方常見問題尋找答案；若問題仍未排除，可透過公開問題回報頁聯絡我們。

常見問題

好記需要註冊或連上網路嗎？

不需要。公開版採純本機模式，不必建立帳號，日常記帳、預算、負債、報表、OCR 與備份都可離線使用，目前不提供 iCloud 同步。

如何建立完整備份？

開啟好記的「設定」，在「資料與備份」選擇「匯出好記備份」，再將副檔名為 .haoji-backup 的檔案保存到你信任的位置。備份可能包含敏感財務資料，請勿放在公開資料夾。

如何還原備份？

在「設定」的「資料與備份」選擇「匯入好記備份」，選取檔案後先檢查格式與筆數，再依畫面選擇合併或取代。完整取代前，好記會先在裝置內建立復原副本並要求再次確認。

如何匯出或匯入交易 CSV？

在「設定」的「資料與備份」使用「匯出交易 CSV」或「匯入交易 CSV」。匯入時會先顯示欄位、筆數與參照預覽，確認後才寫入。跨幣、調整與完整帳務關聯請使用好記完整備份保存。

如何刪除帳務或全部本機資料？

單筆交易可在「流水」中刪除，相關分錄與收據附件會一併移除。部分負債付款或仍被其他資料參照的項目，需要在對應流程處理。

目前沒有「清除全部」按鈕。若要移除 App 中的全部本機資料，請從 iPhone 刪除好記。先前匯出到「檔案」App或其他位置的備份與 CSV，仍需自行刪除。

為什麼收據或發票辨識結果不完整？

影像角度、反光、摺痕、字體與列印品質都可能影響離線辨識。請讓收據平整、文字清晰並避免陰影；也可改從相簿選擇較清楚的照片。

商家、日期、金額、發票號碼與品項都必須在確認畫面人工核對。離線結果未經財政部線上驗證，不能用於確認發票真偽、作廢狀態或完整明細。

如何設定負債與推估利率？

在「資產」分頁新增負債，輸入目前未償本金、剩餘期數、固定月繳額、計算基準日與下次繳款日。好記會依固定利率、月底等額本息模型反推約略利率。

結果是依輸入條件估算，不等同銀行契約的總費用年百分率；若總繳款不足以償還本金，系統會要求修正資料。

桌面小工具為什麼沒有顯示金額？

為降低桌面資訊外洩風險，小工具預設不把實際金額寫入共享快照。到好記「設定」的「隱私與安全」開啟「桌面小工具顯示金額」即可切換。開啟後，能看見桌面的人也可能看到本月摘要。

沒有收到負債或週期交易提醒怎麼辦？

先到好記「設定」確認提醒權限，再到 iOS「設定」檢查好記的通知是否允許。提醒由裝置本機排程；專注模式、通知摘要或系統設定可能影響顯示時間。

Face ID 或 App 鎖定無法使用怎麼辦？

好記會使用裝置可用的 Face ID 或裝置密碼驗證。請先確認 iPhone 已設定裝置密碼，並在 iOS「設定」允許好記使用 Face ID。若系統暫時無法驗證，可依畫面改用裝置密碼或稍後重試。

仍需要協助

回報問題

請提供 iPhone 型號、iOS 版本、好記版本、重現步驟與畫面上顯示的錯誤文字。問題回報會建立在公開的 GitHub Issues，請勿附上實際帳務金額、帳號、收據影像、發票號碼或其他個人資料。
